# HabitFlow

生活習慣をif-thenルールでつなぎ、実行記録とstreak（連続達成）を管理するワークフローツール。

## 技術スタック

- **Frontend**: React + TypeScript
- **ビルドツール**: Vite
- **ワークフローUI**: React Flow
- **スタイリング**: Tailwind CSS
- **状態管理**: useState / useReducer
- **データ永続化**: JSON Export/Import（ローカルファイル）

## 機能

### ワークフローエディタ
- トリガーノード（習慣、時間、場所、感情、イベント）の追加
- アクションノード（実行する習慣）の追加
- ノード間の接続によるif-thenルールの構築

### 習慣トラッキング
- 各アクションの実行記録（「やった」ボタン）
- Streak（連続達成）の計算と表示

### ゲーミフィケーション
- 経験値（XP）システム
- レベリング
- バッジ獲得

### データ管理
- JSONファイルとしてワークフローをエクスポート
- JSONファイルからワークフローをインポート
- データベース不要（すべてJSONで管理）

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```
